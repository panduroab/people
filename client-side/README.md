# People Application

# Dependencies
- Node.js
- Npm

# Install and Run

1. Enter to the root path of the client-side project. 
    ```
    $ cd client_side
    ```
2. Install the node modules.
    ```
    $ npm install
    ```
3. Add your api host in your `.env` file.
    ```
    $ echo 'REACT_APP_API_HOST=http://localhost:3001' >> .env
    ```
4. Run the project.
    ```
    $ npm start
    ```
